# Bitcoin Price Prediction using Machine Learning

This project uses historical Bitcoin price data to predict the next day's closing price using a Linear Regression model. The dataset includes features such as Open, High, Low, Close, Adjusted Close, and Volume. The project demonstrates the complete machine learning workflow, including data preprocessing, visualization, model training, evaluation, and future price prediction.

The model was trained on over 2,700 historical Bitcoin records and achieved an R² score of 0.9894, indicating strong predictive performance. This project was developed using Python and popular data science libraries such as Pandas, NumPy, Matplotlib, and Scikit-Learn.

## Key Features

* Data cleaning and preprocessing
* Exploratory data visualization
* Linear Regression model training
* Model evaluation using MAE and R² Score
* Next-day Bitcoin price prediction
* Model saving using Joblib

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Joblib
